
# Amazon Product Scraper

Scrapes product data from the first page of Amazon based on a search keyword.

## 💻 Backend (Bun)

### Requisitos
- [Bun](https://bun.sh/)

### Como rodar
```bash
bun install
bun index.ts
```

## 🌐 Frontend (Vite)

```bash
cd frontend
npm install
npm run dev
```

Acesse via `http://localhost:5173`
